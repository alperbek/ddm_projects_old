Face Recognation Sample
  • This  is a simple Face recognation program.  uses Eigenface metod
  • Adapted from article "Seeing with OpenCv part 5 Implementing EigenFace"  by Robin Hewitt
  • Servo magazine 05 - 2007
  • this program also includes Fl_OpenCv 0.85
  • Fl_OpenCv is a widget which is render OpenCv image within the FLTK window
  • FLTK is a cross-platform C++ GUI toolkit
  •  Programmed by BlueKid
  • http://derindelimavi.blogspot.com/
How to Use
  1. press "Open-Web-CAM" button
  2. Select a Face by using "Extract Face" Button
  3. Write a name for the selected Face
  4. press "Save and Add Face"  button so Face saved and added to face List
  5. if you want to add new face then go to step 2
  6. after faces selected and saved  press "Learn  Faces"
  7. For recognize a Face  choose a Faces by using "Extract Face" Button
  8. press "Recognize Face" Button
